# Music

Курсовой проект по предмету .NET Framework.
Название проекта: "Музыкальный WEB проигрыватель "Music Collection""
Проект разработан с использованием технологии разработки web-приложений ASP.NET Core
Проект состоит из главной страницы, на которой располагаются пять вкладок (Music Collection, Home, Add Track, Play List, Privacy) и форма для поиска артиста в интернете. 
После ввода в форму поиска название артиста, для осуществления поиска можно нажать мышкой на кнопку Search, либо на клавишу Enter на клавиатуре. 
После этого откроется страница Search с название артиста, количеством результатов поиска и с элементами поиска, которые состоят из обложки альбома, названия артиста, названия альбома, audio тега и кнопки Track List. 
При нажатии на Track List открывается страница с треками альбом, постером исполнителя и названием артиста и альбома. При этом этот альбом добавляется на главную страницу в раздел Recent musics, который может располагать в своём разделе до 4-х альбомов. 
Навигация по страницы Search осуществляется при помощи переключении страниц на панели пагинации или при помощи scroll прокрутки. Возврат на главную страницу можно осуществить при нажатии на Music Collection, либо на Home. 
Для поиска артистов в интернете используется сторонний WEB API.
При нажатии на вкладку Add Track открывается страница для добавления трека в play list. 
В котором имеется форма для заполнения названия трека, артиста, альбома, самого трека и обложки (не обязательно). 
Если форма обложки не будет загружена, то приложение установит обложку по умолчанию. Данные заносятся в базу данных (БД) с помощью Entity Framework. 
При нажатии на кнопку ADD данные заносятся в БД MusicDb и отображаются на страницы  Play List.
При нажатии на вкладку Play List открывается страница Play List. В которой имеются элементы считанные с БД MusicDb. 
Элементы состоят из обложки, audio тега, названия артиста, трека и альбома. На обложке элемента имеется кнопка в виде мусорного ведра для удаления трека из Play List и БД MusicDb. 
Соответственно для проигрывания трека нужно нажать на аудио тег.
При нажатии на вкладку Privacy открывается страница с данными о приложении.
В проекте используются два контроллера. Один для работы с внешним web API (HomeController). Второй для работы с БД (MusicController). 
Для оформления и стилизации приложения используется bootstrap и jquery.
